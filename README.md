# 말차 (Matcha)

React 기반의 말차 투어소개개 페이지 프로젝트입니다.

## 배포

- **프로덕션 도메인**: [`https://matcha-rho.vercel.app/`](https://matcha-rho.vercel.app/)
- **플랫폼**: Vercel

## 기술 스택

- **프론트엔드**: React 18 (Create React App)
- **스타일/애니메이션**: AOS, `lottie-react`, `react-icons`, `typed.js`
- **번들러/도구**: `react-scripts`


## 로컬 실행 방법

1. 저장소 클론
   ```bash
   git clone <this-repo-url>
   cd Matcha
   ```

2. 의존성 설치
   ```bash
   cd tour
   npm install
   ```

3. 개발 서버 실행
   ```bash
   npm start
   ```

4. 브라우저에서 확인
   - 기본 주소: `http://localhost:3000`

## 빌드

```bash
cd tour
npm run build
```
